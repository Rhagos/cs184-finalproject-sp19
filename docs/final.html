Final Project Milestone

Gun Range Simulation


Benjamin Wu, Daniel Wu, Charles Hager

Project Background:

This VR application brings players to a simulated gun range where they can use a variety of firearms with several different attachments.
Firearm training is inaccessible to some people due to factors such as location, cost, hassle, and safety, all of which is compounded for novice shooters.
A virtual reality gun range is a great first step for those who want to learn to handle firearms, as it is a risk-free and beginner-friendly environment to practice.
Furthermore, virtual reality is a growing field, and projects like this can be extended for use in video games or more complex simulations

Abstract:

The Virtual Reality Gun Range is a virtual reality project that puts the user into a virtual reality gun range, where the user can then practice aiming and firing with a variety of firearms. In addition to the traditional virtual reality experience, an inverse kinematics arm position solver was implemented in order to enhance the virtual reality experience and provide more immersion. The virtual reality itself was implemented with Unity, and features fully functional firearms that the user can interact with using the Oculus controllers, where interaction is firing, aiming, picking up, dropping, and reloading. Arm position is then calculated with inverse kinematics given the head and two hands, which is then portrayed in VR.

Technical Aspects:

The Virtual Reality was implemented with Unity, and is described here
The Inverse Kinematics was implemented by interpolating the arms from 3 points, and the details can be found here

Some main issues encountered were the conversion of in-game coordinates to actual movement coordinates, and the offset between the hands and arms, since the sensors model the palms, resulting in some odd clipping between the built in hands and the arms that we constructed.
Overall, we learned that it is best to start early and that some things don't need to be perfect, and we also figured out how to implement solvers into a VR space.
