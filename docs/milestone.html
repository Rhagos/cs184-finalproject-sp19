Final Project Milestone

Gun Range Simulation


Benjamin Wu, Daniel Wu, Charles Hager

Project Summary:

This VR application brings players to a simulated gun range where they can use a variety of firearms with several different attachments.
Players can customize the range environment, targets, and other parameters to suit their preferences and needs.
Firearm training is inaccessible to some people due to factors such as location, cost, hassle, and safety, all of which is compounded for novice shooters.
A virtual reality gun range is a great first step for those who want to learn to handle firearms, as it is a risk-free and beginner-friendly environment to practice.

Current Accomplishments:

  • Preliminary environment scene
  • Pistol model + animations
  • Physical interactions with the pistol (collisions, gravity)
  • Script for firing and raycasting bullets
  • Functional gun range target placeholders
  • Pistol firing and proper animation interactions
  • Significant progress on inverse kinematics

Preliminary results:

  • Player can pick up and fire a pistol at targets using any hand.
  • Targets react properly upon being hit (placeholder targets)
  • Inverse Kinematic solver has 2D functionality with arm-hand movement. Hand rotation to be worked out later.
  • Basic gun functionality:
    • Firing and playing animations are implemented
    • Other interactions (reloading, chambering, changing firing mode) are not yet implemented.
